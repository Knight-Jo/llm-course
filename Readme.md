针对古典诗歌生成任务，基于Qwen-0.6B基础模型进行监督微调(SFT)，实现根据给定作者、标题、标签生成符合格式要求的唐诗。本实验验证LoRA微调方法在特定领域任务中的有效性。

### 2. 数据筛选标准
| 维度       | 标准                      | 过滤比例 |

| 作者       | 2-15个有效汉字            | 3.2%     |

| 诗句       | 4-16行，每行5-35字        | 12.7%    |
| 标签       | 至少包含1个有效标签        | 2.5%     |
| 总字数     | 20-500字                  | 8.3%     |
